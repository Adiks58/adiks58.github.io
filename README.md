# Toyota Trueno AE86 (Hachiroku) – информационный сайт

Проект представляет собой статический информационный сайт, посвящённый культовому автомобилю Toyota AE86 (Hachiroku). Сайт состоит из трёх страниц:

- **История** – рассказ о создании, культурном значении и роли AE86 в дрифт-движении, дополненный карточкой с общими техническими данными.
- **Технические характеристики** – детальное описание двигателя 4A-GE, системы T-VIS, габаритов, подвески, трансмиссии и динамических показателей.
- **Спорт и покупка** – информация о гоночных достижениях (дрифт, ралли, кольцевые гонки), а также актуальные цены на вторичном рынке России и Японии.

## Цель проекта

Создать удобный информационный ресурс для энтузиастов японского автопрома, фанатов дрифт-культуры и потенциальных покупателей Toyota AE86. Сайт систематизирует исторические, технические и рыночные данные об автомобиле.

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, адаптивность)
- JavaScript (ES6) – переключение страниц без перезагрузки

## Инструкция по запуску

1. Убедитесь, что все файлы проекта (включая папку `Новая папка` со скриншотами) находятся в каталоге `C:\Сайт`.
2. Откройте файл `index.html` в любом современном браузере (двойным щелчком или через меню браузера).
3. Для корректной работы локальных изображений не требуется интернет.

## Системные требования

- Любая операционная система (Windows, macOS, Linux, iOS, Android).
- Современный браузер: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari (последние версии).
- Разрешение экрана – от 320 пикселей (сайт адаптивен).
